# Team Logo Instructions

## How to Add the Team Logo

1. **Prepare your logo image:**
   - Format: PNG, JPG, or SVG
   - Recommended size: 512x512 pixels for best quality
   - Background: Transparent (PNG) or matching the theme colors
   - Colors: Should match Palestinian flag theme (Black, White, Green, Red)

2. **Add the logo file:**
   - Save your logo as: `assets/images/team_logo.png`
   - Supported formats: PNG, JPG, JPEG, SVG

3. **The logo will automatically appear in:**
   - Home screen team card (60x60 pixels)
   - Team screen header (80x80 pixels)
   - Both with Palestinian flag gradient border
